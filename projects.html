Projects

Below are detailed descriptions of selected projects I completed during college and internships.

DESIGN OF ADAPTIVE SLIDING MODE CONTROL FOR EVs WITH HYBRID ENERGY STORAGE

Developed an adaptive sliding-mode controller to manage power distribution between battery and hybrid storage (battery + supercapacitor). The controller adapts online to changing vehicle load and improves transient response and battery life. The project covered mathematical modeling, simulation in MATLAB/Simulink, design of power interface, and prototype testing with microcontroller-based hardware-in-the-loop.

Key topics: Control systems, power electronics, EV drive, MATLAB/Simulink, microcontroller prototype

Efficient IoT-Based Landslide and Flood Alert System

Designed a sensor network to monitor water levels, rainfall and soil moisture. Microcontrollers gather sensor data and send to a cloud service. Threshold-based algorithms detect hazard conditions and trigger SMS/notification alerts to authorities and residents.

Key topics: IoT, sensors, GSM alerts, data logging

Green House Monitoring System

Built an embedded system to automatically adjust LED lighting depending on temperature and grow conditions, and provided manual override modes for users. Implemented using Arduino and sensor feedback.

Key topics: Embedded systems, sensors, automation

Automatic Street Light Control System

Implemented an LDR-based control to switch lights ON at low ambient light and OFF during daylight — a simple energy-saving solution deployed as a small-scale prototype.

Key topics: Sensors, energy saving, microcontroller
